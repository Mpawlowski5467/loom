# Alice Chen

## Role

Frontend lead on the [[loom-knowledge-graph]] project. Responsible for the graph visualization layer, Sigma.js integration, and the Plate editor.

## Background

- 6 years of experience in data visualization and interactive web applications
- Previously built network analysis dashboards at a cybersecurity firm
- Strong background in D3.js, Three.js, and WebGL
- Advocate for [[local-first-software]] — introduced the team to CRDTs and Automerge

## Current Focus

- Implementing the force-directed layout with breathing animations in Sigma.js
- Building the node hover/select interaction model (highlight neighbors, dim others)
- Exploring camera utilities for smooth zoom-to-node transitions
- Investigating [[graph-databases]] query patterns to optimize graph.json loading for large vaults

## Communication Style

Prefers async communication. Shares detailed write-ups in project threads rather than real-time chat. Appreciates when meeting notes are thorough and linked to relevant topics.

## Key Contributions

- Designed the color system for node types (blue for projects, green for topics, purple for people)
- Proposed the bidirectional sync between file tree and graph view
- Built the initial prototype of the wikilink autocomplete in the editor
